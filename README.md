# Open-University-Learning-Analytics-dataset

Kuzilek, J., Hlosta, M. & Zdrahal, Z. Open University Learning Analytics dataset. Sci Data 4, 170171 (2017). [link](https://doi.org/10.1038/sdata.2017.171)
